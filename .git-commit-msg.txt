feat(ui): 대시보드/CTA 전 페이지에 AmbientBackdrop 글래스모피즘 배경 적용

문제
- /dashboard 와 모든 CTA 서브페이지(/camping, /cine-trip, /trekking,
  /photo-gallery, /wellness, /medical-tourism, /audio-guide, /crowd-radar,
  /related-spots, /pet-travel, /dashboard/images)의 배경이 까만 단색에 가까워
  로그인/회원가입/지원 페이지의 글래스모피즘 무드와 비교해 단조롭다는 피드백
- 기존 radial-gradient 도 투명도가 0.08~0.18 수준이라 시각적 깊이가 부족
- 추가로 카드 → 상세 → 뒤로가기 전환에서 페이지 톤이 매번 다르게 느껴짐

설계
- 21st.dev 「Geometric」(회전 타원형 블러 도형) 패턴을 inspiration 으로 받아,
  로그인의 원형 orb 와는 형태가 다른 회전 타원형 + 부드러운 드리프트로 차별화
- 페이지마다 팔레트만 바꿔 같은 컴포넌트로 정체성 유지

신규 컴포넌트
- netplix-frontend/components/AmbientBackdrop.js
  · 5 개의 회전 타원형 블러 도형(60px blur)이 framer-motion 으로 12~16s 주기
    수직 드리프트 + mount-in 페이드/회전 애니메이션
  · position: fixed + z-index: -1 → 부모(`isolation: isolate`) 의 스태킹
    컨텍스트 안에서 콘텐츠 뒤로 안전 격리, pointer-events: none 으로 인터랙션
    절대 차단하지 않음
  · 상단 오로라 글로우 + 하단 페이드(가독성 보강) + intensity/topGlowColor 옵션
  · theme: "dark" | "light" — 라이트 배경 페이지(pet-travel)에서는 하단 페이드를
    흰색 베일로 자동 전환
  · palette 4~5 개 색상을 받아 도형마다 다른 색 부여

페이지별 적용 (각각 다른 팔레트로 정체성 유지)
- dashboard:        violet · pink · blue · amber · cyan
- camping (list+detail): green · emerald · amber · cyan
- cine-trip:        rose · amber · indigo · sky
- trekking:         sky · amber · green · violet
- photo-gallery:    pink · violet · amber · cyan
- wellness:         emerald · violet · sky · pink
- medical-tourism:  sky · violet · amber · cyan
- audio-guide:      violet · cyan · pink · amber
- crowd-radar:      cyan · indigo · pink · emerald
- related-spots:    indigo · pink · emerald · amber
- dashboard/images: rose · violet · amber · cyan (로딩/에러/정상 3 분기 모두)
- pet-travel:       amber · pink · violet · cyan (light 모드, intensity 0.7)

추가 개선
- 각 페이지의 root div 에 `position: relative` + `isolation: isolate` 부여 →
  AmbientBackdrop 의 z-index:-1 이 페이지 밖으로 새지 않도록 스태킹 컨텍스트 형성
- 기존 radial-gradient base 의 채도/투명도를 0.18~0.22 로 살짝 올리고
  third color stop 을 추가해 베이스 자체도 덜 단조롭게 보강
- camping list/detail 의 hero/main 클래스에 명시 z-index:1 추가
