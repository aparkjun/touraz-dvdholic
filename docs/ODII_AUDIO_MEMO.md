# Odii 오디오 가이드 — 참고 메모

## 건수 인지 차이 (한국어 기준)

- **기대**: 오디오 샘플(콘텐츠)이 **6,000개 넘게** 있다고 알고 있음.
- **관측**: 앱/API에서는 **한국어(ko) 기준 약 2,200건+** 수준만 노출된 경험이 있음.

## 코드베이스 맥락 (조사용)

- `VisitKoreaOdiiHttpClient` 주석에 **themeBasedList 총건수 약 2,231**, **storyBasedList 는 공공 API 기준 6,000건 초과**라고 적혀 있음 → 탭/타입(THEME vs STORY)과 공공 API `totalCount`·풀 적재 여부를 구분해 볼 필요가 있음.
- 실제 건수는 **언어별 카탈로그**, **호출 한도(429)**, **캐시·페이지 상한** 등에도 영향받을 수 있음.

## 후속 (내일)

- 한도 리셋 후 Odii 재호출·화면 건수 재확인.
- 필요 시 THEME/STORY 각각 공공 API `totalCount`와 앱 로드 건수를 대조.
